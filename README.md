# SIXTy Seconds

AI-powered rental car check-in that replaces decision fatigue with smart, personalized recommendations in under 60 seconds.

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion + Rive
- **Payments**: Stripe + Apple Pay
- **API**: SIXT API integration
- **PWA**: Progressive Web App with iOS optimization

## Features

- **Smart Recommendations**: AI-driven decision tree based on trip context (destination, passengers, weather, luggage)
- **Quick Questions**: Swipeable card interface for rapid preference gathering
- **Lucky Spin**: Gamified upgrade offers with Rive-powered fortune wheel
- **Apple Pay**: Full payment flow integration in PWA
- **Native Feel**: Proper iOS safe area handling and mobile viewport optimization
